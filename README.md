# 🎵 Spotify Clone

A front-end clone of Spotify built using **HTML5** and **CSS3**. The project recreates the look and feel of Spotify's interface to improve frontend development skills.

## ✨ Features

- Responsive music player UI
- Sidebar navigation
- Playlist section
- Album and artist cards
- Hover animations
- Clean and modern design

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Purpose

This project was developed to practice frontend development by recreating a popular music streaming platform interface using only HTML and CSS.

- Music playback
- Search feature
